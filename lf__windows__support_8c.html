◆ lf_clock_gettime()

int lf_clock_gettime ( instant_t *  t)

Fetch the value of _LF_CLOCK (see lf_windows_support.h) and store it in tp. The timestamp value in 'tp' will always be epoch time, which is the number of nanoseconds since January 1st, 1970.

Returns
0 for success, or -1 for failure. In case of failure, errno will be set to EINVAL or EFAULT.
